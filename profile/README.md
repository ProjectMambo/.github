A design-driven ecosystem of optimized Unix configurations, unified palettes, and automated dotfile deployment architectures.

Portfolio: [kohkohnut.org](https://kohkohnut.org/) 
Wiki: [projectmambo.org](https://projectmambo.org/)

## The Ecosystem
 - **[MamboDot](https://github.com/ProjectMambo/MamboDot)** - Our desktop configuration repository containing automated installation deployments for Hyprland, Waybar, Kitty, and much more.
 - **[MamboFolio](https://github.com/ProjectMambo/MamboFolio)** - Portfolio of the author of **Project Mambo**.
 - **[MamboSite](https://github.com/ProjectMambo/MamboSite)** - Documentation regarding project details and usage for the ecosystem.
 - **[MamboColour](https://github.com/ProjectMambo/MamboColour)** - The core, centralized design colour palette for consistent styling across the ecosystem.
 - **[MamboFont](https://github.com/ProjectMambo/MamboFont)** - The font design for consistent styling and spacing across the ecosystem.

## Community & Contributions
Since this is our personal organisation, we are not looking for external pull requests. However, if you spot a bug or have a suggestion regarding the features or scripts, feel free to open an **Issue** to let us know:
 * **Reporting Design Issues:** If a colour token breaks contrast in an application theme, please open an issue directly inside the MamboColour repository.
 * **Feature Suggestions:** For custom script additions, keybinding defaults, or other features, target the specific repository issue board.
